# AI Meme Coin Launchpad ($0 Deploy)

## 1. Deploy to Vercel
1. Push this repo to GitHub
2. Go to [vercel.com](https://vercel.com) → Import Project
3. Select your repo → Deploy (auto)

## 2. Setup Free Tiers
- **thirdweb**: [dashboard.thirdweb.com](https://thirdweb.com) → Create app → Get Client ID
- **Lemon Squeezy**: [lemonsqueezy.com](https://lemonsqueezy.com) → Create $19 product → Copy link
